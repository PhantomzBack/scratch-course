Why Scratch?

Here's Why!





Scratch is a powerful drag and drop tool, created by MIT in collaboration with Google, which is efficient and helps lay the basic foundation for coding. This can help children develop their coding logic and thinking skills. It's intuitive drag and drop design makes it very easy for children to write simple logic programs. Click on the "Why Scratch" button above to check out Scratch yourself.





Scratch is not only designed for coding, but various other compuer science fields fall under the category. Scratch can be used for producing games, animations, music with embedded tools like TTS(Text-to-Speech), translate, video-sensing, etc. Apart from that, Scratch is compatible with various hardware tools like Makey Makey, LEGO Mindstorms EV3 and many more!





The Scratch community is an always growing community where enthusiasts create fascinating projects for others to see. Not only can others see the project, but remix it and try to make it better, which is almost a system like cloning and editing in Github. If you ever face any issues, Scratch has a very active forum and documentation and to help creators even more. And don't worry, if you don't wish to show your project to the world, you can keep it private.
